You are designing a set of marketplace graphics for EssentialsX Dashboard, a premium web control panel that lets Minecraft server owners manage the EssentialsX plugin from the browser. These graphics will sell the product on the BuiltByBit marketplace, so they must look polished, trustworthy, and high-end — the visual quality should signal "professional paid software," not a free hobby plugin.

Critical rule: match the real product. Do not invent a fantasy UI. Use the exact brand system below so the graphics look like authentic screenshots and promo art of the actual app.

=== BRAND SYSTEM (use these exactly) ===
- Primary background: very dark slate #020617. Card/surface: #0f172a. Elevated surface: #1e293b. Borders: #334155.
- Accent (the hero color): coral-red #e13c43, with a lighter #f45e65 for glows/highlights on dark. Use accent sparingly and deliberately — buttons, active nav, key numbers, a single focal glow. Everything else stays cool slate.
- Text: near-white #f1f5f9 (headings), #cbd5e1 (body), #94a3b8 (muted/labels).
- Status colors: success green #15803d / #86efac, danger #b91c1c / #fca5a5.
- Typography: Inter for all UI and marketing text (tight, modern, geometric sans). Monospace (Consolas/Menlo style) for console/code/YAML snippets.
- Shapes: generous rounded corners (cards ~12px radius), flat surfaces with very subtle shadows and 1px slate borders. No skeuomorphism, no heavy gradients except tasteful dark-to-darker backdrops and one soft coral glow as a focal accent.
- Overall mood: sleek enterprise-SaaS dashboard (think Linear / Vercel / Stripe dashboard quality) but for a Minecraft audience. Dark theme is the primary canvas.

=== REALISTIC UI MOTIFS TO FEATURE (these are real features — depict them accurately) ===
- A left sidebar with grouped nav items and an active item marked by a coral left-border bar + faint coral tint.
- A sticky topbar with a search/command-palette pill (showing "⌘K"), a notification bell, a theme toggle, and a circular avatar.
- Card-based content: stat cards with a big number + tiny sparkline trend; a live console panel with monospace log lines (color-coded info/warn/error) and a command input; an economy table with player avatars and balances; a moderation list with status badges; a player inventory grid of Minecraft item slots.
- Minecraft player avatars (blocky head icons) and item icons appear inside the otherwise-SaaS UI — this is the one place the gaming context shows through. Keep it subtle and classy.
